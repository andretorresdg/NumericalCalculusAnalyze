ep2-numerico-versao4.py


Este exercício programa apresenta como função auxiliar o aprofundamento do entendimento dos estudante sobre métodos numéricos para possibilitar sua aplicação em problemas reais. Desta forma, este programa traz métodos que possibilitaram a resolução de problemas com o equacionamento da temperatura em uma barra. Neste âmbito, buscamos utilizar valores pré-definidos de variáveis para desenvolver resultados satisfatórios graficamente e numericamente ao usuário.


Para executar o projeto, será necessário:


Como baseamos nosso programa em um arquivo presente na página https://edisciplinas.usp.br/course/view.php?id=74332, o usuário deverá usar o arquivo desenvolvido especificamente para este exercício programa. Assim, o documento de texto presente deve ser baixado e seu caminho, consecutivamente, copiado e colado quando pedido pelo programa.


Possuir um executor do programa que seja igual ou superior ao Python 3.8.2. (recomendo a utilização do Virtual Studio Code como um Python IDE).
Possuir uma biblioteca matplotlib e outra numpy


Para utilizar os programas corretamente, além do usuário ter de baixar o executor do programa através do site oficial, também deverá importar as bibliotecas matplotlib.pyplot, numpy, random e math. Isso é feito através do prompt do python, com os comandos de “pip install” seguido pelo nome da biblioteca.
